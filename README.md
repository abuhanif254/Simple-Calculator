# Calculator Project

A simple web-based calculator built with HTML and CSS.

## Features
- Basic arithmetic operations: addition, subtraction, multiplication, division
- Responsive design
- User-friendly interface

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/abuhanif254/Simple-Calculator.git
   ```
2. **Navigate to the project folder:**
   ```bash
   cd project_calculator
   ```
3. **Open `index.html` in your browser.**

## Project Structure
- `index.html` - Main HTML file for the calculator UI
- `style.css` - Stylesheet for the calculator
